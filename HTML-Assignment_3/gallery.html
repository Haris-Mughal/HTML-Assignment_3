HTML5: Figure and Figure Caption

<figure></figure>

Images often come with caption. HTML5 has introduced a new element to contain images and their caption so that the two are associated. You can have more than one image inside the element as long as they all share the same caption.

<figure></figure>

The element has been added to HTML5 in order to allow web page auther to add a caption to an image. Before these elements were created there was no way to associate an element with the caption. Older browsers that do not understand HTML5 element simply ignore the elements and display the content of them.
